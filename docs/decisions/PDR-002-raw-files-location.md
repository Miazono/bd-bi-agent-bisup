# Decision
Загрузка через путь raw/hm/filename/date/filename.csv т.к. Hive external tables читают все файлы из указанной директории как одну таблицу.

# Purpose
Исключит смешивание файлов с разной схемой в одном raw-префиксе.
Упростит загрузку `raw -> bronze`, проверку наличия файлов и повторные запуски по `load_date`.
Упростит дальнейшее масштабирование, если появятся новые источники или новые даты загрузки.
