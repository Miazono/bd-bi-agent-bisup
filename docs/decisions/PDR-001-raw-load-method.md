# PDR-001: Способ загрузки raw-слоя

## Контекст
Raw-слой проекта загружается отдельным Python-скриптом, который уже умеет считать MD5, проверять существование объекта в MinIO и повторно запускаться без лишней перезагрузки одинаковых файлов.

## Решение
Использовать `ingestion/load_raw.py` как основной механизм загрузки raw-файлов в MinIO. Скрипт размещает файлы в объектном пути с `load_date`, что делает загрузку воспроизводимой и позволяет безопасно повторять запуск для одной и той же партии данных.

## Последствия
Повторные прогоны не перезаписывают неизменённые файлы, а новые версии можно отличить по дате загрузки и хэшу. Это упрощает контроль целостности raw-слоя и дальнейшую загрузку в bronze без ручной сверки.
