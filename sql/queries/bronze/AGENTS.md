# AGENTS.md — sql/queries/bronze

## Назначение
В этом каталоге лежит SQL-логика преобразований, используемая для загрузки bronze-таблиц из внешних raw-таблиц.

## Область
- `INSERT INTO ... SELECT ...` запросы для загрузки bronze
- приведение типов и нормализация пустых значений рядом с источником
- технические поля загрузки вроде `ingest_ts`, `source_file_name`, `batch_id`

## Правила
- Не держи в bronze-запросах тяжёлую бизнес-логику.
- Предпочитай один основной запрос загрузки на каждую bronze-таблицу.
- Плейсхолдеры параметров должны оставаться простыми и явными, например `__BATCH_ID__`.
- Операционная оркестрация, небольшие проверки и сильно динамический SQL должны оставаться в Python-лоадерах.
