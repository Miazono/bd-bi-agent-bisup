# AGENTS.md — sql/queries/silver

## Назначение
В этом каталоге лежит SQL-логика преобразований, используемая для построения silver-таблиц из bronze и справочных таблиц.

## Область
- `DELETE`, `INSERT` и `MERGE` запросы для обновления silver
- переиспользуемый аналитический SQL с явно указанным grain таблицы
- сложные преобразования, которые проще ревьюить как SQL, чем как строки Python

## Правила
- Оркестрацию, маршрутизацию чанков и небольшие проверки держи в Python.
- Предпочитай простые токены-плейсхолдеры вроде `__BATCH_ID__`, `__PREFIX_LEN__`, `__PREFIX_VALUE__`.
- По возможности держи в одном файле один основной логический шаг.
- Если меняется физическая схема, обновляй `sql/ddl/silver/` и документацию вместе.
